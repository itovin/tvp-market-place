# TVP Marketplace

A full-stack web marketplace application for our subdivision, designed to let homeowners list and share products instead of relying on messaging platforms where posts quickly get buried.

This project is currently in early development.

## Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Spring Boot
- Java
- Maven

## Current Features

- User registration (frontend + backend integration)

## Upcoming Features

- Login system using Spring Security and JWT authentication

## Project Status

This project is actively being developed. Future updates will include authentication, user profiles, product listings, and marketplace functionality.
